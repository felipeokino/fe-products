## Run Locally  
Clone the project  

~~~bash  
  git clone https://github.com/felipeokino/fe-products.git
~~~

Go to the project directory  

~~~bash  
  cd fe-products
~~~

Install dependencies  

~~~bash  
(pnpm | npm | yarn) install
~~~

> **Note**
> Create .env file following env.example
>

Start the server  

~~~bash  
(pnpm | npm | yarn) start
~~~
